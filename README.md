# calculator

순수 자바스크립트를 이용해서 만드는 계산기

### 진행 과제
- 입력창(결과창), 임시 결과창, 계산 과정창
- . 중복 체크
- 숫자와 기호 비교
- AC 버튼 [전체 초기화]
- 각 기호별 계산
- **+@ 키보드 입력**

계산기 디자인은 아이폰 계산기 참고
